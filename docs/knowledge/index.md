# Canonical Knowledge Registries

Shared knowledge is defined once here and referenced by flows. The table lists exactly the registries present in this directory.

| Registry | Purpose | Owner |
|----------|---------|-------|
| [Domain](domain.md) | Shared business concepts, invariants, state models, and policies | Domain / Business Owner |
| [Contracts](contracts.md) | Interface identity, ownership, versions, and compatibility | Backend Team / Data Platform Team |
| [Database](database.md) | Conceptual data model, persistence model, ER, indexes, relationships, constraints | Data Platform Team |
| [Glossary](glossary.md) | Concise shared business vocabulary | Documentation Steward |
| [References](references.md) | Curated external and internal links | Documentation Steward |
